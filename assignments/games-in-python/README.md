
# 📘 Assignment: Games in Python

## 🎯 Objective

Build a simple Hangman game using Python so you can practice loops, conditionals, string operations, and user input handling.

## 📝 Tasks

### 🛠️ Build Core Hangman Gameplay

#### Description
Create the main game loop for Hangman where a player guesses letters to reveal a hidden word.

#### Requirements
Completed program should:

- Randomly choose a word from a predefined word list.
- Display the current word progress using underscores for unknown letters.
- Prompt the user to guess one letter at a time.
- Reveal correctly guessed letters in all matching positions.

### 🛠️ Track Attempts and End Conditions

#### Description
Add game rules for incorrect guesses and finishing the game with clear feedback.

#### Requirements
Completed program should:

- Track the number of incorrect guesses remaining.
- Prevent duplicate guesses from reducing attempts multiple times.
- End the game when the word is fully guessed or attempts run out.
- Print a win or lose message that includes the final word.
